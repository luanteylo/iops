## TOTO Overhead Analysis on PlaFRIM

**Machine:** PlaFRIM (Inria/University of Bordeaux, France)
**Study:** Runtime overhead of the [TOTO](https://gitlab.inria.fr/hpc_io/toto) I/O interception library
**Parameters swept:** Process count, I/O request size, access pattern, file strategy, analysis period
**Metrics:** IOR bandwidth (MiB/s), total execution time, CPU and memory overhead

This study measures the performance impact of intercepting I/O system calls via `LD_PRELOAD` across a range of IOR workloads on BeeGFS. The report includes bandwidth comparisons between baseline and instrumented runs, overhead heatmaps, and resource utilization analysis.

[**Open the full interactive report**](toto-overhead-plafrim.html)

## Submit Your Own Report

Have an interesting benchmark study powered by IOPS? We welcome community contributions. To feature your report here:

1. Generate your report with `iops report`
2. Open an issue on the [IOPS repository](https://gitlab.inria.fr/hpc_io/iops/-/issues) with a brief description of the study
3. Attach or link the HTML report file

We will review and add it to this page.
